Weather Portal

Discover Weather Portal: Your ultimate destination for up-to-the-minute weather updates. Harnessing the power of the National Weather Service API and OpenStreetMap Foundation's mapping technology, Weather Portal provides a seamless experience across seven dynamic pages. From site information to radar imagery, easily access detailed weather forecasts, alerts, and location-specific data. With an intuitive interface and reliable sources, Weather Portal elevates your weather tracking journey.

  • Multi Webpage weather service tool for real-time weather data with the following features:
    • User-friendly interface for easy navigation and accessibility to weather information
    • Integration with mapping API from OpenStreetMap Foundation for precise location information
    • Seven pages providing comprehensive information:
      • Home: Establish site's goals and information
      • About: Gives Background and credits services utilized
      • Weather: Gets Weather Information Based off of requested location
      • Alerts: Displays all active weather alerts
      • Radar: Displays active radar from the National Weather Service
  • Designed using Visual Studio Code
    • Programing Languages: JavaScript
    • Markup: HTML
    • Sheet Style: CSS
    • API Services:
      • Weather Information by the National Weather Service
      • Mapping by the OpenStreetMap Foundation

Mobile Application Item Database

Meet M.A.I.D.: A dynamic Mobile App that redefines product management. With a user-centric design, it offers robust features. Access a comprehensive database with key attributes like ID, Name, Quantity, and more. Seamlessly navigate Display, Modify, and Deletion Menus. Adding new products is intuitive.
