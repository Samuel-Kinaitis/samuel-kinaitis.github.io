Classy Closet

Experience Classy Closet: A visionary eCommerce mock-up that reimagines online shopping. Seamlessly navigate categories, explore trending items, and personalize your choices. Manage your cart, modify the database, and streamline records efficiently. With local storage and supporting pages, Classy Closet offers an elevated adventure.

  • Multi Webpage eCommerce mock-up with the following features
    • A responsive navigation menu
    • A home page with category selections, featured products, and treading items.
    • A catalog that is both categorically sortable by item type and searchable by name keywords.
    • A product page to view selected items and select options such as the size or amount
    • A dynamic shopping cart that allows for items to be removed with checkout total
    • A dynamic database that can be modified to manage, remove, and input new items into the site’s catalog.
    • Employee section to manage database
    • Locally stores the user's cart to remember upon a revisit.
    • Supporting pages such as contact, TOS, careers, and support
  • Designed using Visual Studio Code
    • Programing Languages: JavaScript
    • Markup: HTML
    • Sheet Style: CSS
    • Storage - Cookies (First-Party) Local Storage and IndexedDB.

Mobile Application Item Database

Meet M.A.I.D.: A dynamic Mobile App that redefines product management. With a user-centric design, it offers robust features. Access a comprehensive database with key attributes like ID, Name, Quantity, and more. Seamlessly navigate Display, Modify, and Deletion Menus. Adding new products is intuitive. The Mobile Application Item Database ensures an efficient, all-encompassing experience.

  • Mobile Application for managing a database of products with the following components:
    • Database that stores Product: ID, Name, Expiration Date, Manufacture, Quantity, Sale Price, Cost, Description
    • Display Menu that displays all products from the database in a table
    • Modification Menu that allows for items to be searched for and changed
    • Deletion Menu that allows for items to be searched and removed
    • Add Menu to add new products to the database
  • Designed using Android Studio
    • Programing Language: Java
    • Markup: XML
    • Storage: SQLite

Employee Record Database

Introducing a robust Desktop program for seamless employee record management. Easily organize, add, edit, and delete entries in spreadsheet style. Enjoy flexible column sorting, quick changes, and efficient keyword searches. Elevate your record-keeping with this user-friendly desktop solution.

  • Desktop program for managing employee record database with the ability to:
    • Display in spreadsheet style
    • Organize columns from least to greatest or vice versa
    • Add new items
    • Edit pre-existing items
    • Save changes to database
    • Delete selected items
    • Undo previous deleted actions
    • Remove all entries
    • Search database with keyword
  • Designed using Visual Studio 2022 and MS Access
    • Programing Language: C#
    • Storage: Microsoft Access
